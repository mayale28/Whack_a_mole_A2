The two files in this folder are designed to be extended for 202 coursework Assignment 2 - Whack-a-mole Game. 

 - StartGame.html: Opens a browser window of the specified size.
 - GameLogic.html: The file that will contain all of the game logic, target style sheets, and page formatting. 

We have already provided some target stylesheets for you (circleCentral, circle10px ..etc) but feel free to add your own! 

To start the game, open the StartGame file. This will open a browser window of a specific size where the game shall run. 

If the game doesn't open when you click the StartGame.html file your browser might be BLOCKING pop-ups. 

In this case, just disable blocking pop-ups for the StartGame.html webpage in your browser.
